# VLAN Separation Lab

This lab demonstrates VLAN separation using one Cisco switch and four PCs.

## VLAN Design

- VLAN 10: Admin
- VLAN 20: User

## Devices Used

- 1 Cisco 2960 switch
- 4 PCs

## Result

- Devices in the same VLAN can communicate.
- Devices in different VLANs cannot communicate without inter-VLAN routing.
